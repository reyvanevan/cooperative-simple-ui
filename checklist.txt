=== NOTES ===
- 3 DISTINCT SIMPANAN TYPES (each member has separate balance for each):
  * POKOK = mandatory one-time deposit (100k-500k), NEVER withdrawable, lifetime
  * WAJIB = mandatory monthly deposit (50k/month), withdrawable ONLY when exit koperasi
  * SUKARELA = voluntary anytime deposit, can withdraw freely anytime, any amount
- TOTAL SIMPANAN = POKOK balance + WAJIB balance + SUKARELA balance  
- Each type tracked separately in database & UI
- Tier system based on POINTS (not simpanan): BRONZE (0-999), SILVER (1000-2999), GOLD (3000-5999), PLATINUM (6000+)
- Nomor anggota format: MBR-YYYYMMDD-XXX
- Points earned from purchases: Rp 1000 spent = 1 point
- Points: Rp 1000 = 1 point (customizable)
